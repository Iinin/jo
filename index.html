Build great apps with less code

Jo is a lightweight library for making modern web apps.

  • Makes fast, single-page apps
  • Develop locally with your favorite editor
  • Use your favorite build tools
  • Made with pure JavaScript and CSS
  • Open source (BSD-3 license)
  • Small footprint

Project Status

Jo was first released in April, 2010, and since then it's had over 40,000 downloads. Work is underway for version 1.0 (we're skipping from 0.5.0 because... well because).

We're looking for help with:

  • platform testing
  • example apps
  • tutorials
  • community management
  • making it easier for other developers to extend widget classes
